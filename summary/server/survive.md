# 生存服务器

{% hint style="info" %}
此服务器版本为1.17.1-\[支持最高稳定版本]
{% endhint %}

## 服务器补充规则 <a href="fu-wu-qi-bu-chong-gui-ze" id="fu-wu-qi-bu-chong-gui-ze"></a>

{% hint style="danger" %}
1. 反高频:我们允许每秒闪烁 **3** 次红石信号
2. 高空流水:需限制在 **20** 格以内且有容器接住(最好类似水池、河流)
3. 需从主城开始向外建造(尽力吧)
{% endhint %}
